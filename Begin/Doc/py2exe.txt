 чтобы скомпилировать py в exe, надо ввести команду в командную строку на установку pyinstaller и wheel 
 
 pip install pyinstaller
 pip install wheel 
 После этого снова заходи в командную строку и пиши pyinstaller "здесь указывай полный путь до твоего проекта, но без русских букв" 
 Через некоторое время в папке dist, твоего имени пользователя появится папка с подпапками. Там ищи exe файл. И ещё. После pyinstaller можешь сбить
 --onedir это у тебя будет один, файл вместо кучи папок с файлами.﻿
